#### Mplayer 
![mplayer](screenshots/mplayer.png?raw=true)

```
# Clone
git clone https://github.com/talesluna/floyd-player-golang/ && cd floyd-player-golang

# Install go
sudo apt install golang

# Config GOPATH
export GOPATH="../youPath"

# Download packages
go get github.com/golang/gxui

# Run 
go run *.go

# Or build
go build 
```
